# Clave del Día - Instrucciones de Descarga

## Opción 1: Acceder a la Aplicación Web (Recomendado - Funciona en Todos los Dispositivos)

La aplicación está disponible como una **Progressive Web App (PWA)** que funciona como una app nativa en dispositivos móviles.

**Enlace**: [https://013a92af213c4e46bbbabd59724c01e9-29831dc092a4419f8abd0c7aa.fly.dev/](https://013a92af213c4e46bbbabd59724c01e9-29831dc092a4419f8abd0c7aa.fly.dev/)

### Instalación en Android:
1. Abre el enlace en Chrome o navegador móvil
2. Toca el menú (⋮) y selecciona "Instalar aplicación" o "Agregar a pantalla de inicio"
3. Elige un nombre y toca "Instalar"
4. La aplicación se instalará como una app nativa

### Instalación en iOS:
1. Abre el enlace en Safari
2. Toca el botón "Compartir" (cuadro con flecha)
3. Desplázate hacia abajo y toca "Agregar a la pantalla de inicio"
4. Toca "Agregar" en la esquina superior derecha
5. La aplicación se instalará en tu pantalla de inicio

## Opción 2: Descargar APK (Solo Android)

Si prefieres una aplicación nativa de Android:

1. Asegúrate de tener **Android 7.0 o superior**
2. En tu dispositivo Android, ve a **Configuración > Seguridad** y activa "Instalar de fuentes desconocidas"
3. Descarga el archivo APK desde el servidor
4. Toca el archivo descargado para instalar

## Características

- **Período de Prueba**: 15 días desde la descarga/instalación
- **Funcionalidad Offline**: La aplicación funciona sin conexión a Internet
- **Fecha de Expiración**: Después de 15 días, la aplicación se bloqueará

## Ciclo de Vida del Período de Prueba

- **Fecha de Instalación**: Se registra automáticamente en tu dispositivo
- **Contador**: Cuenta regresiva de 15 días
- **Bloqueo**: Después del día 15, verás una pantalla de expiración

## Soporte

Si tienes problemas con la instalación o uso, por favor:
1. Asegúrate de tener conexión a Internet para la descarga inicial
2. Borra la caché del navegador si experimentas problemas
3. Reinicia tu dispositivo si la aplicación no se instala correctamente

## Seguridad

- La aplicación no requiere permisos especiales
- No recopila datos personales
- Todos los cálculos se realizan localmente en tu dispositivo
- Los datos de instalación se almacenan solo en tu dispositivo
